# Client Server Application using SFTP and Java
Transfer Protocol: RFC 913 (https://tools.ietf.org/html/rfc913)

## Client - Server Features

| COMMAND | Client | Server |
| :---         |     :---:      |          ---: |
| USER   | Y     | Y    |
| ACCT   | Y     | Y    |
| PASS   | Y     | Y    |
| TYPE   | Y     | Y    |
| LIST   | Y     | Y    |
| CDIR   | Y     | Y    |
| KILL   | -     | -    |
| NAME   | -     | -    |
| DONE   | Y     | Y    |
| RETR   | -     | -    |
| STOR   | -     | -    |
